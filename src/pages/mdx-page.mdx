**Requesitos Previos**



Astro se basa en Node.js, por lo que necesitarás tenerlo instalado en tu sistema.

**Instalación de Astro:**

Abre tu terminal o línea de comandos.
Ejecuta el siguiente comando para instalar Astro de forma global:

```js
('npm install -g astro')
```

**Verificación de la instalación:**

Después de la instalación, verifica que Astro se haya instalado correctamente ejecutando:

```js
('astro -v')
```

**Configuración inicial:**

Puedes inicializar Astro en tu proyecto ejecutando:

```js
('astro init')
```

**Uso básico:**

Explora los comandos básicos de Astro, como astro dev para iniciar un servidor de desarrollo y astro build para compilar tu proyecto.

**Personalización del proyecto:**

Edita los archivos en tu proyecto según tus necesidades. Puedes modificar el archivo astro.config.mjs para ajustar la configuración.
